# ErenshorRU — Русификатор Erenshor

Мод для русификации [Erenshor](https://store.steampowered.com/app/2382520/Erenshor/) — однопользовательской RPG в стиле классических MMORPG.

## Что переводится

- Интерфейс, меню, настройки
- Описания классов, статов, навыков и заклинаний
- Квесты и диалоги NPC (100% покрытие)
- Предметы и лор
- Чат SimPlayer'ов и системные сообщения
- Русский ввод в игровой чат (NPC и SimPlayer'ы распознают русские команды)
- Кликабельные ключевые слова в диалогах NPC (с обратным маппингом на английские keywords)

Файлы переводов лежат в папке `Translation/` в формате `английский=русский` — легко править и дополнять.

## Установка

1. Установить [BepInEx 5.x](https://github.com/BepInEx/BepInEx/releases) в папку игры
2. Запустить игру один раз, чтобы BepInEx создал структуру папок
3. Скачать [последний релиз](https://github.com/sitxovski/ErenshorRU/releases/latest) (`ErenshorRU.zip`)
4. Распаковать архив в `<папка игры>/BepInEx/plugins/`
5. Запустить игру

Структура должна выглядеть так:

```
Erenshor/
└── BepInEx/
    └── plugins/
        └── ErenshorRU/
            ├── ErenshorRU.dll
            ├── Fonts/
            │   ├── Montserrat-Regular.ttf
            │   ├── Montserrat-Bold.ttf
            │   ├── ... (все варианты Montserrat)
            │   └── PxPlus IBM VGA9.ttf
            └── Translation/
                ├── ui_system.txt
                ├── ui_extra.txt
                ├── misc.txt
                ├── quests.txt
                ├── items_lore.txt
                ├── skills_spells.txt
                ├── simplayer_chat.txt
                ├── npc_dialogs.txt
                ├── npc_keywords.txt
                └── tutorials.txt
```

## Обновление

Мод автоматически проверяет наличие новой версии при запуске игры. Если доступно обновление — в игре появится уведомление с описанием изменений.

## Как работает

- **Harmony-патчи** перехватывают `TMP_Text.text` и `Text.text`, подставляя русский перевод на лету
- **Шрифты Montserrat** заменяют игровые шрифты с полной поддержкой кириллицы и динамическим атласом глифов
- **PxPlus VGA9** заменяет пиксельный шрифт PerfectDOS для консольного стиля
- **Стем-маппинг NPC-ключевых слов** (`npc_keywords.txt`) обеспечивает подсветку и кликабельность переведённых ключевых слов в диалогах
- **Обратный маппинг** преобразует нажатия на русские слова обратно в английские keywords для корректной работы квестовой логики
- **Русский ввод** в чат автоматически обогащается английскими ключевыми словами, чтобы NPC и SimPlayer'ы понимали команды

## Как помочь

1. Форкните репозиторий
2. Отредактируйте нужный файл в `Translation/`
3. Создайте Pull Request с описанием изменений

Или просто создайте [Issue](https://github.com/sitxovski/ErenshorRU/issues) с описанием проблемы.

## Сборка из исходников

Требуется .NET SDK и установленная игра.

1. Открыть `ErenshorRU.csproj`
2. Поправить путь `GameDir`, если игра стоит не в стандартной папке Steam
3. `dotnet build -c Release`
