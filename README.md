# ErenshorRU — Русификатор Erenshor

Мод для русификации [Erenshor](https://store.steampowered.com/app/2382520/Erenshor/) — однопользовательской RPG в стиле классических MMORPG.

## Что переводится

- Интерфейс, меню, настройки
- Описания классов, статов, навыков и заклинаний
- Квесты и диалоги NPC (100% покрытие)
- Предметы и лор
- Чат SimPlayer'ов и системные сообщения
- Русский ввод в игровой чат (NPC и SimPlayer'ы распознают русские команды)

Файлы переводов лежат в папке `Translation/` в формате `английский=русский` — легко править и дополнять.

## ⚠️ Важно

Перевод выполнен с помощью ИИ и **не тестирован на 100%**. Возможны:

- Неточности в переводе отдельных фраз
- Текст, который может «криво» выглядеть в интерфейсе (обрезаться, не помещаться)
- Пропущенные строки, которые мы не обнаружили

**Будем рады любой помощи!** Если нашли ошибку или хотите улучшить перевод — создавайте Pull Request или Issue на [GitHub](https://github.com/sitxovski/ErenshorRU). Файлы переводов — обычные текстовые файлы, редактировать их может каждый.

## Установка

1. Установить [BepInEx 5.x](https://github.com/BepInEx/BepInEx/releases) в папку игры
2. Запустить игру один раз, чтобы BepInEx создал структуру папок
3. Скачать [последний релиз](https://github.com/sitxovski/ErenshorRU/releases/latest)
4. Скопировать папку `ErenshorRU` целиком в `<папка игры>/BepInEx/plugins/`
5. Запустить игру

Структура должна выглядеть так:

```
Erenshor/
└── BepInEx/
    └── plugins/
        └── ErenshorRU/
            ├── ErenshorRU.dll
            └── Translation/
                ├── ui_system.txt
                ├── ui_extra.txt
                ├── misc.txt
                ├── quests.txt
                ├── items_lore.txt
                ├── skills_spells.txt
                ├── simplayer_chat.txt
                └── npc_dialogs.txt
```

## Сборка из исходников

Требуется .NET SDK и установленная игра.

1. Открыть `ErenshorRU.csproj`
2. Поправить путь `GameDir`, если игра стоит не в стандартной папке Steam
3. `dotnet build -c Release`

## Как работает

- Harmony-патчи перехватывают `TMP_Text.text` и `Text.text`, подставляя русский перевод на лету
- Кириллица отображается через системный шрифт Segoe UI (fallback)
- Русский ввод в чат автоматически обогащается английскими ключевыми словами, чтобы NPC и SimPlayer'ы понимали команды
